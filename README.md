# Clone Clubhouse
![wakatime](https://wakatime.com/badge/github/MaksymMakiaveli/clone_clubhouse.svg)
###  Stack
Front-end
   * ReactJS/NextJS
   * TypeScript
   * Redux
   * Axios
   * Socket.IO
   * SASS
  



## Getting Started

First, run the development server:

```bash
git clone https://github.com/MaksymMakiaveli/clone_clubhouse.git
```
```bash
npm i
```
```bash
npm run dev
```
